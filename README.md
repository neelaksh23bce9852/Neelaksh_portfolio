# Manda Neelaksh | AI Developer Portfolio

A modern, clean, and professional developer portfolio for Manda Neelaksh, an AI enthusiast and builder of intelligent systems.

## Tech Stack

- **Framework:** Next.js 16 (React 19)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Theme:** Minimal Dark with Mesh Gradients

## Key Sections

- **Hero:** Impactful introduction with smooth animations.
- **About:** Personal journey and focus on practical AI solutions.
- **Projects:** Featured AI-powered tools (EcoLens, Gravity Intelligence, AuditAI, etc.).
- **Skills:** Core competency grid with technical stack.
- **Experience:** Professional internship and hackathon activities.
- **Contact:** Clean call-to-action for collaboration.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
